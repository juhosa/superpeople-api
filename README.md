# SuperPeople API

One endpoint returning an array of superhumans.

## `GET /get-superpeople`

Returns

```json
[
  {
    "name": "Batman",
    "superpower": "Martial arts",
    "color": "#000000"
  },
  {
    "name": "Spiderman",
    "superpower": "Spideysense",
    "color": "#a71814"
  },
  {
    "name": "Gamora",
    "superpower": "Accelerated healing factor",
    "color": "#9aac00"
  }
]
```
